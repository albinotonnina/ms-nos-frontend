# MSNOS-frontend

A backbone applicaton for the project Ms-nos, a simple network operating system for Microservices
<https://github.com/workshare/ms-nos>

### on Heroku
<https://ms-nos.herokuapp.com>


##Development instructions

```
$ npm install
$ bower install
$ grunt serve
$ grunt watch
```

###Dist

```
$ grunt production
```

####Node app
port 5000

```
npm start
```
